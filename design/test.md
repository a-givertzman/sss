# Общие данные
Тестовые расчеты предназначены для проверки ПО при ежегодном, промежуточном и возобновляющем освидетельствовании на борту судна в присутствии инспектора РС или в другом необходимом случае. 
Для каждого тестового расчета приведены результаты расчета с использованием ПО и сравнение значений с приведенными в Документации по остойчивости на судно. Погрешность определяется по формуле:
> % = (базовое значение – рассчитанное значение) / допустимое значение х 100

В качестве базового значения принимается значение из одобренной Документации по остойчивости. В случае указано два значения погрешности допустимой погрешности, в качестве допустимой величины принимается большее из них.
